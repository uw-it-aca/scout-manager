Noise Levels

Definition:

The amount of noise that is generally socially acceptable in the space.

Format:

Silent

Use when whispering is a noticeable sound in the space, such as in a silent study room.

Quiet (Displays to students as "low hum")

Use when it is acceptable in the space to type, shuffle papers, and hold low conversations, but not to converse at normal volume. A student called this "being able to hear other people think."

Chatter

Use when it is acceptable in the space to converse at normal volume.

Variable

Use when the noise level in the space changes throughout the day. Use for all outdoor spaces.

When not to use it:

Do not list a noise level for study rooms (enclosed areas designed for one person or group), classrooms, or other spaces that are self-contained and in which the noise level of a room is determined by the current occupant(s).

How the filter works:

Spaces listed as "variable" will show up if the user searches for low hum or chatter. Variable spaces will not show up when users search for silent.

Spaces with no noise level listed (mostly study rooms and classrooms) will show up regardless of the user's search, since the noise level of the room is determined by the current occupant(s).

Capacity

Definition:

The number of seats in the space.

Format:

Number

When not to use it:

Never use it for spaces that are "open space" or "outdoor area" types. Also, don't use it if you have "computer lab," "lounge," "study area," or "conference/classroom" spaces that have a capacity greater than 20.

How the filter works:

If the student performs a search for a room of x capacity, all rooms that are x capacity or greater will be returned. Spaces with no capacity listed will show up for any search (since there, x=0). This means that students looking for enough space for just a few students will still see these large or open areas in their search results, since they are appropriate spaces for small groups. (It also means you don't have to get an accurate count of seats in large areas.)

Space Types

Definition:

The category or categories (up to two) that best describe the selected space.

Format:

Study room

Enclosed space for individual or group

Study area

Unenclosed space for individual or group

Computer lab

Group of computer stations; number of computers grouped together should equal something students would call a “lab”

Production studio

Work space with specialized equipment or technology, such as a multi-media studio

Conference/classroom

Larger-capacity rooms primarily used for classes, meetings, or workshops

Open space

Area for use by multiple individuals or groups; may contain a range of furniture -- carrels, tables, lounge furniture

Lounge

Indoor casual space, typically with lounge furniture and low tables

Café

Space that functions primarily as an eatery

Outdoor area

Outdoor casual space

Location Description

Format:

In general, the location description should start with the most broad information ("Suzzallo Library") and end with the most precise ("Room 201," "NW corner," etc.). Separate each with a comma. The minimum is the building name.

Room Number:

Include the room number only if the physical space is labeled with this number. If the number is only used on floor plans, students will be unlikely to associate it with the space itself.

Avoid redundancy:

If the space name is "2nd floor study area," then the location description doesn't need to include "2nd floor." If the space name is "Room 201" then the location description shouldn't include "Room 201." If the space name contains enough information to find it, the location description might just be the name of the building.

Examples from UW Seattle:

  • Space name: Lobby Area; Location description: Fishery Sciences, Main floor
  • Space name: Orin's Place; Location description: Paccar Hall, 2nd floor
  • Space name: Team Room 214; Location description: Paccar Hall
  • Space name: Commons Room 103; Location description: Alder Hall Commons, NE 40th St entrance
  • Space name: T5 Cubbies (5); Location description: Health Sciences T Wing, 5th floor, North
  • Space name: EAL Reading Room; Location description: East Asia Library, Gowen Hall, Room 322
